Parameters
nNumber of observations.
[in]inputPointer to an array of length n, containing the values to be filtered.
[in]blockPointer to an array of length n, containing the block of origin for each cell. Each entry should be less than the mm.size() in the constructor.
[out]outputPointer to an array of length n, to store the high-quality calls (see the other filter() overload). If overwrite = false, entries are only set to false for low-quality cells; this effectively performs AND'ing of high-quality calls across multiple calls to filter().
overwriteWhether to overwrite existing false-y entries in output.
