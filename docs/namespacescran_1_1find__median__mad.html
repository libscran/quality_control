Detailed Description

Calculate the median and MAD from an array of values.

Pretty much as it says on the can; calculates the median of an array of values first, and uses the median to then compute the median absolute deviation (MAD) from that array. We support calculation of medians and MADs for separate "blocks" of observations in the same array.
